MySWG/Markdown language

MySWG uses the standard markdown syntax, as defined at markdown

To be more precise, MySWG uses René Jeschke's Txtmark that presents some (small) differences with respect original syntax.

Furthermore,

  • MySWG admits macros as described in MMD.
  • MySWG accepts HTML tables.
  • Lines starting with '//' are skipped.
  • [](URL) works as [URL](URL)
  • The pattern 'áéíóú' in the first line detecs input file encoding (e.g. UTF-8)
